1. Parameters are the values passed into a function or method, while Arguments are the values that are specified when the function is invoked. Example:
static void Main (string[] args)
{ SayHi("someName"); // An argument
Console.ReadLine(); }
static void SayHi(string name) // A parameter { Console.WriteLine("Hello " + name); }

    Number 5

    Arithmetic overflowing happens when the value passed inside of a data type exceed its size of storage. For example: The data type 'byte' can only take a maximum number of 255, if we try passing a number larger than its storage size, it returns 0 meaning an overflow
